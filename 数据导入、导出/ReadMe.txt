1、创建逻辑目录  用system用户登录
create directory dpdata1 as 'd:\test\dump';

2、创建对应的实际目录 'd:\test\dump'，把导出文件dmp.DMP拷贝到创建的文件夹dump中

3、查看是否存在
select * from dba_directories;

4、system用户登录，给特定用户赋予在制定目录的操作权限（按用户导出时使用）
例如给hospital用户授权：grant read,write on directory dpdata1 to hospital;


5 导入命令:
rem 整个数据库导入
rem impdp system/tigersysdba DIRECTORY=dpdata1 DUMPFILE=HIS环县20160704.DMP FULL=y logfile=impdp_His.log;
rem 按用户导入
rem impdp system/tigersysdba schemas=hscenter DIRECTORY=dpdata1 DUMPFILE=hscenter.dmp  logfile=impdp_hscenter.log;
rem impdp system/tigersysdba schemas=hsemr    DIRECTORY=dpdata1 DUMPFILE=hsemr.dmp     logfile=impdp_hsemr.log;

6、导出命令：
rem 整个数据库导出
rem expdp system/tigersysdba DIRECTORY=dpdata1 DUMPFILE=full.dmp FULL=y;
rem 按用户导出
rem expdp system/tigersysdba@his schemas=hscenter DIRECTORY=dpdata1 DUMPFILE=hscenter.dmp  logfile=expdp_hscenter.log;
rem expdp system/tigersysdba@his schemas=hsemr    DIRECTORY=dpdata1 DUMPFILE=hsemr.dmp     logfile=expdp_hsemr.log;
